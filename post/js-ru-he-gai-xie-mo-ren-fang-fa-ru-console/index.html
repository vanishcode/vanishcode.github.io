js如何改写默认方法（如console）

2018-11-06 1 min read

demo：改写console

console.log = (function (origin) {
    return function (str) {
        origin.call(console, "hello:" + str);
    }
})(console.log);
console.log("name");

注意这个origin参数就是原来的函数。之前没想明白，改写属性可以，改写方法的话（尤其追加一些操作）原方法怎么执行呢？然后问了一下别人，google了一下，特地记录。
