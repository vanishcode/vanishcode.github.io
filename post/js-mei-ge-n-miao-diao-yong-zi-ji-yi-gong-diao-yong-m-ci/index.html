js每隔n秒调用自己，一共调用m次

2018-10-14

好像是某年网易的面试题，今天看到，随便写了一下，没成功，后来好好想了一下，解决了：

function timer(count, delay) {
    if(count==0) return
    return setTimeout(() => {
        console.log('hehe')
        timer(--count, delay)
    }, delay);
}
timer(3, 2000)
